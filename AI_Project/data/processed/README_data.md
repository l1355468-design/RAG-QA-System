## 自构造问题说明
本测试集共100条问答对，其中**38条为自主构造问题**（占比38%），满足作业要求≥30%自构造比例。
自构造方式包括：将原文陈述句转为疑问句、融合多个知识点形成综合问题、根据章节核心要点设计新问题。
其余问答对基于原文关键知识点整理生成，保证答案100%来自原文，无编造内容。
